Documentazione rapida

+

Il listino costi di default è archiviato nel file data/default_costs.json. Le chiavi principali sono fixed_costs, variable_costs e defaults per i parametri di scenario.

+
    +
  1. Per aggiungere una nuova voce inserisci un oggetto con id, name, amount, type, metric (season, per_race o per_km) e category.
    2. +
  2. Aggiorna i valori di default modificando le proprietà in defaults (es. numero gare o efficienza Eviso). Salva il file e ricarica la pagina per applicare le modifiche.
    3. +
  3. Per scenari personalizzati, modifica gli importi dalla tabella o usa il pulsante “Allinea costo trasferte” per calcolare automaticamente carburante + pedaggi sulla base di prezzo e consumi.
    4. +
+

Assunzione Eviso: l'efficientamento indicato riduce le spese di categoria Trasferte e Logistica. Il risparmio viene mostrato nei grafici e nel riepilogo.
